二、配置Mask系统默认模板

1、mask="Dec"，小数保留2位，不使用分隔符

    <span id="input4" uitype="Input" mask="Dec" maskoptions="{'precision':2, 'separator':''}" ></span>
例：

2、mask="Money"，要以"￥"为前缀

    <span id="input5" uitype="Input" mask="Dec" maskoptions="{'prefix':'￥'}" ></span>

二、自定义模板

1、mask="Custom"，自定义手机模板输入

    <span id="input6" uitype="Input" mask="Custom" maskoptions="{'model':'19999999999', 'placeholder':'_'}" ></span>
例：

2、mask="Custom"，自定义通配符，#代表1至9，不包含0

    //定义通配符
    comtop.UI.InputMask.definitions['#'] = '[1-9]';

    <span id="input7" uitype="Input" mask="Custom" maskoptions="{'model':'#9%', 'placeholder':'_'}" ></span>
