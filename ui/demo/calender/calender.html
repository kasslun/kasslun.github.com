三、日期范围限制（只支持date模式）

1、设置最小，最大可选范围（通过日期设置限制）

        <span id="calender70" uitype="Calender" mindate="2012-10-22" maxdate="2013-02-11"></span>

例：

2、设置最小，最大可选范围（通过表达式设置限制）

        //如果最小最大日期不是静态的，而是根据当前日期变化而变化，那么可以使用表达式设置范围
        //假如要设置最小日期为三天前，最大日期为下一个月的今天的前两天
        <span id="calender71" uitype="Calender" mindate="-3d" maxdate="+1M,-2d"></span>

例：

3、设置最小，最大可选范围（通过表达式设置限制，更为复杂的）

        <span id="calender72" uitype="Calender" mindate="-2y,+2M,-3d" maxdate="+3y,+1M,-2d"></span>

例：

七、回调设置

1、选择日期后回调（单日期）

        <span id="calender24" uitype="Calender" on_change="calCallBack"></span>
        function calCallBack(date){
            alert('返回的值是个数组：' + date);
        }

例：

2、选择日期后回调（范围）

        <span id="calender26" uitype="Calender" isrange="true" on_change="calCallBack2"></span>
        function calCallBack2(date, date1){
            alert('date：' + date + '\n' + 'date1：' + date1);
        }
