常用配置项

1、宽度设置，width的值可为数字或百分比如："200px"或"50%"

        <span id="calender11" uitype="Calender" width="300px" ></span>
例：

2、可读设置，设置readonly

        <span id="calender13" uitype="Calender" readonly="true" value="2012-12-21" ></span>
例：

3、文本显示设置，设置textmode

        <span id="calender14" uitype="Calender" textmode="true" value="2012-12-21"></span>
例：

4、触发方式设置，设置trigger，值可为"click"/"mouseover"

        <span id="calender12" uitype="Calender" trigger="mouseover" ></span>
例：

5、使用确定按钮，设置okbtn，当okbtn == true，点选日期需要点击确定按钮后才能输出至输入框

        <span id="calender15" uitype="Calender" okbtn="true"></span>
例：

6、设置星期天为一周的第一天，设置sunday_first（只对date功能模式有效）

        <span id="calender16" uitype="Calender" sunday_first="true"></span>
例：

7、设置输入框为空时的提示文字（则placeholder），设置emptytext

        <span id="calender16" uitype="Calender" emptytext="请选择发工资日期..."></span>
