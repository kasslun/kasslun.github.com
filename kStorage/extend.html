本地自动完成

这是一个cStorage的自动完成扩展组件，主要用以显示当前域的输入的历史记录，对当前页面的所有输入框生效。

请在下面输入框中输入一些内容，失去焦点后，再次输出的时候，将提示输入记录。

用法

1.本组件没有任何接口，只需引用kStorage.autocomplete.min.css和在kStorage后面引用kStorage.autocomplete.min.js便可。

<!DOCTYPE html>
<html>
    <head>
        <link rel="stylesheet" href="extend/kStorage.autoComplete.min.css"/>
    </head>
    <body>
        ...
        ...
        <script src="kStorage.min.js" type="text/javascript" ></script>
        <script src="extend/kStorage.autoComplete.min.js" type="text/javascript" ></script>
    </body>
</html>

2.对disabled、readonly或有属性autocomplete="off"的input不生效，密码输入框不会产生自动完成。

    <!-- 如下类型的输入框不生效 --<
    <input readonly="readonly" type="text" />
    <input disabled="disabled" type="text" />
    <input autocomplete="off" type="text" />
    <input type="password" />

3.设置如下属性kst.autocomplete = "off"，当前页面的所有input都不会产生自动完成。

4.它将记录你的输入内容到本地存储，只记录2-20个字符的输入内容。
